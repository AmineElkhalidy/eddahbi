# Dr. Youssef Eddahbi Clinic Website

This project is a modern, responsive, and multilingual landing page for **Dr. Youssef Eddahbi (General Practitioner)**.

It presents the clinic with a clean and trustworthy design, includes key sections for visitors (hero, about, services, and contact), and supports **Arabic, French, and English** with proper right-to-left and left-to-right layouts.

The website is built to provide a smooth user experience across mobile and desktop, with clear navigation, localized content, and direct appointment/contact access.
